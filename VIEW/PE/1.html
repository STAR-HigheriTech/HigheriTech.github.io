中文显示问题解决方案

如果页面中文显示异常（乱码），请尝试以下解决方案：

  1. 确保文件以UTF-8编码保存
  2. 检查服务器是否设置了正确的Content-Type头部：Content-Type: text/html; charset=utf-8
  3. 如果使用Apache服务器，可在.htaccess中添加：AddDefaultCharset UTF-8
  4. 如果使用Nginx服务器，可在配置中添加：charset utf-8;

入门指引

欢迎使用 FirePE 文档中心。这里您可以找到所有有关安装、使用和定制 FirePE 的信息。

安装与自动加载机制

  • 放置插件
    直接将下载好的 .2% 或 .1gm 文件放到程序目录，无需编程。
  • 启动后扫描
    WinPF 启动后，会扫描 Resources 和 input 目录，按文件名顺序依次加载所有插件包。
  • 加载效果
    - EdgeHex 插件：自动加载并注入新的文件系统，可添加工具、替换界面资源等。
    - HPM 模块：通过 HotPEModeLib 引擎加载，支持添加工具、加载驱动等功能。

快速开始

按照以下步骤快速开始使用 FirePE：

  1. 从官方网站下载最新版本的 FirePE
  2. 使用提供的工具将 FirePE 写入 USB 驱动器
  3. 从 USB 驱动器启动计算机
  4. 开始使用 FirePE 进行系统维护或故障排除
# FirePE 启动命令示例
firepe --boot --device /dev/sdb1 --format NTFS

什么是 WebPE？

WebPE 是 FirePE 的在线组件，允许用户通过网络访问额外的工具和资源，无需完全下载到本地。

主要特性：

  • 云端工具库，随时获取最新工具
  • 轻量级设计，减少本地存储占用
  • 自动更新，始终保持最新版本
  • 跨平台兼容，支持多种设备访问

为什么选择 FireF7

FireF7 是 FirePE 的核心组件，提供了高效的启动管理和系统维护功能，具有以下优势：

  • 快速启动时间 优化
  • 广泛的硬件支持 兼容
  • 直观的用户界面 易用
  • 丰富的工具集 功能全面

如何使用启动器进入WebPE

要使用启动器进入 WebPE，请按照以下步骤操作：

  1. 启动 FirePE 系统
  2. 从主菜单选择 "WebPE 启动器"
  3. 等待启动器初始化完成
  4. 选择您需要的工具或功能
  5. 开始使用 WebPE 功能
